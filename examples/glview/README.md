Simple OpenGL viewer for glTF geometry.

## Requirements

* premake4 : Requires recent `premake4` for macosx and linux, `premake5` for windows.
* GLEW
* glfw3

### MacOSX and Linux

    > premake4 gmake
    $ make

### Windows(not tested)

    > premake5.exe vs2013
    Open .sln in Visual Studio 2013
